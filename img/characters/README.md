# キャラクター立ち絵

ここに PNG / WEBP 画像を置くと自動的に尋問画面の立ち絵として使用されます。

| ファイル名      | キャラ         |
|--------------|--------------|
| sera.png     | セラ（偵察天使）  |
| iris.png     | アイリス（天使騎士）|
| mika.png     | ミカ（副隊長）   |
| kronos.png   | クロノス（大天使長）|

- 推奨サイズ: 512×768 以上（縦長）
- 画像がない場合は絵文字のフォールバックが表示されます
- `aicon/` フォルダの画像をコピーしてリネームするだけで OK です
